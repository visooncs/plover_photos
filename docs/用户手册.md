# Plover Photos 用户手册

欢迎使用 Plover Photos！本文档将指导您如何使用系统的核心功能来管理和享受您的照片库。

## 1. 快速入门

### 访问系统
打开浏览器访问 `http://localhost` (或您服务器的 IP 地址)。

### 界面概览
- **时间轴**: 首页默认展示所有照片，按拍摄时间倒序排列。
- **媒体库**: 包含“人物”、“地点”、“最近添加”等分类入口。
- **地图**: 在地图上查看您的足迹。
- **系统维护**: 管理后台任务、导入照片和查看系统状态。

## 2. 导入照片

Plover Photos 不会移动或修改您的原始文件，它通过“扫描”的方式建立索引。

1. 进入 **系统维护** 页面。
2. 找到 **扫描照片库** 任务。
3. 点击 **运行**。
4. 系统将扫描 Docker 配置中挂载的所有路径（如 `/mnt/d/Photos`）。
   > **注意**: 请确保您的照片文件夹已挂载到 Docker 容器中。

## 3. AI 智能功能

系统内置多种 AI 能力，建议按以下顺序执行初始化：

### 3.1 人脸识别
- **功能**: 自动识别照片中的人脸，并将同一人的照片聚类。
- **操作**: 在系统维护页点击 **人脸识别** -> **运行**。
- **查看**: 完成后，在 **媒体库** -> **人物** 中查看结果。您可以在此为人物命名。

### 3.2 语义搜索
- **功能**: 让您能通过自然语言搜索照片。
- **操作**: 点击 **生成语义向量** -> **运行**。
- **使用**: 在顶部搜索栏输入描述，如：
  - "海边的日落"
  - "穿红衣服的小孩"
  - "雪山"
  - "生日蛋糕"

### 3.3 智能回忆
- **功能**: 生成“那年今日”等主题回忆。
- **操作**: 点击 **生成回忆** -> **运行**。
- **查看**: 首页顶部或侧边栏会显示生成的回忆卡片。

## 4. 浏览与管理

### 查看详情
点击任意照片进入大图模式：
- **信息面板**: 点击 `i` 图标查看详细 EXIF 信息（相机、光圈、ISO、GPS等）。
- **动态照片**: 如果是 Live Photo，鼠标悬停或长按即可播放视频部分。
- **原图下载**: 支持下载原始文件。

### 视频播放
系统支持网页直接播放大多数常见格式的视频。如果不兼容，建议在浏览器中安装解码插件或转码上传。

### 删除照片
- **放入回收站**: 点击删除图标将照片移入回收站，不会立即物理删除。
- **清空回收站**: 在系统维护页运行 **清空回收站** 任务，将永久删除文件。

## 5. 常见问题

**Q: 为什么搜索不到照片？**
A: 请确保已运行“生成语义向量”任务。新导入的照片需要先进行 AI 处理才能被搜索到。

**Q: 地图上没有照片？**
A: 只有包含 GPS 信息的照片才会显示在地图上。请运行“更新位置信息”任务以解析坐标对应的地名。

**Q: 视频里的人脸能识别吗？**
A: 可以。系统会自动抽取视频帧进行人脸检测。请确保运行了最新版本的“人脸识别”任务。
